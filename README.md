# aws-crypto-rust

Rust implementations of AWS cryptographic libraries.

## Structure

- `esdk/` - AWS Encryption SDK for Rust
- `mpl/` - AWS Cryptographic Material Providers Library for Rust
- `prim/` - Cryptographic primitives
- `aws_structured_encryption/` - AWS Database Encryption SDK (structured encryption)
- `db-esdk/` - Database ESDK (future)
- `s3ec/` - S3 Encryption Client (future)
